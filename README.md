# Chest X-ray MLLM Evaluation Prototype

An example website for exploring chest X-ray classification and severity-grading results from cloud-based multimodal large language models, including Claude, GPT, Gemini, and Grok.

Based on the research project:

**“Cross-task Evaluation of MLLMs in Chest X-ray Classification and Severity Grading Under Standard and CoT Prompting”**

Website: https://2i03e2f.github.io/ChestXrayAIPrototype/

## Dataset

* Source: NIH ChestX-ray14
* Total images used in this prototype: 20
* Raw model responses collected from:

  * GPT-5.5
  * Claude Opus 4.7
  * Gemini 3.5 Flash
  * Grok 4 Mini
